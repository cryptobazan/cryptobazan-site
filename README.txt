کریپتیوبازان — بسته‌ی آماده وب‌سایت
===================================

فایل‌ها:
- index.html
- styles.css
- script.js
- assets/ (شامل تصاویر placeholder)

توضیحات سریع:
این بسته یک سایت تک‌صفحه‌ای (static) با تم تیره و نئون برای برند «کریپتیوبازان» است.
برای استفاده، پوشه را روی یک هاست استاتیک آپلود کنید.

نکته: تصویر لوگو و هِرو را در پوشه `assets/` قرار دهید با نام‌های زیر (یا نام دلخواه ویرایش مسیر در index.html):
- assets/logo-placeholder.png  ← لوگوی شما (پیشنهاد اندازه: 200x200)
- assets/hero-placeholder.png  ← تصویر یا اسکرین‌شات نمودار (پیشنهاد اندازه: 1200x800)

چگونه روی GitHub Pages منتشر کنم (پیشنهاد ساده):
1. یک ریپازیتوری جدید در GitHub بسازید (مثلاً `cryptobazan-site`).
2. تمام محتویات این پوشه را در ریپازیتوری کامیت و پوش کنید.
3. به Settings ریپازیتوری بروید → Pages → Branch: `main` و پوشه `/ (root)` را انتخاب کنید و Save کنید.
4. بعد از چند دقیقه سایت شما در آدرس `https://<username>.github.io/<repo>/` قابل مشاهده است.

نشر روی Firebase Hosting (هاست رسمی گوگل):
1. به https://firebase.google.com بروید و یک پروژه بسازید.
2. روی دستگاه خود Node.js نصب کنید و سپس firebase-tools را نصب کنید:
   npm install -g firebase-tools
3. در پوشه پروژه لاگین کنید و init کنید:
   firebase login
   firebase init hosting
   // گزینه‌ی public را روی `public` قرار ندهید — از `./` یا `public` به دلخواه استفاده کنید.
4. فایل‌ها را deploy کنید:
   firebase deploy --only hosting
5. سایت شما روی دامنه Firebase قابل مشاهده خواهد بود و می‌توانید دامنه دلخواه خود (مثلاً example.com) را نیز متصل کنید.

نکات امنیتی و SEO اولیه:
- برای افزایش سرعت، تصاویر را قبل از آپلود فشرده کنید.
- متا تگ‌ها و Open Graph را در head اضافه کنید اگر می‌خواهید اشتراک‌گذاری بهتر در شبکه‌ها داشته باشید.
- اگر نیاز به فرم تماس واقعی دارید، از Formspree، Netlify Forms یا endpoint سرور استفاده کنید.

اگر می‌خواید من این فایل‌ها را برایتان روی GitHub Pages یا Firebase آپلود کنم، راهنمایی گام‌به‌گام و کدهای لازم را می‌دهم.
